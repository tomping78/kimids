    제품소개

    하이브리드 통합을 위한 엔터프라이즈 패턴 구현

    비즈니스에 중요한 매우 안정적이고 확장 가능하며 안전한 서비스 및 애플리케이션을 통합합니다. TIBCO BusinessWorks™ 소프트웨어를 통해 통합 전문가는 마이크로서비스 및 컨테이너로 구축된 최신 클라우드 기반 API 주도 접근 방식과 기존 엔터프라이즈 통합 패턴을 모두 사용하여 애플리케이션 전략을 구현할 수 있습니다. Eclipse를 기반으로 하는 제로 코드 모델 기반 환경은 개발 복잡성을 단순화하여 비용을 줄이고 시장 출시 시간을 단축하는 데 도움이 됩니다.

    TIBCO BusinessWorks™ 소프트웨어는 Integrate의 능력 TIBCO Cloud™ Integration iPaaS(서비스로서의 통합 플랫폼).
    TIBCO Businessworks 애플리케이션을 TIBCO Cloud Integration iPaaS로 이동하여 클라우드로의 이동을 가속화하고 더 많은 사용자가 맞춤형 통합 경험을 통해 디지털 자산을 연결할 수 있도록 지원하십시오.

    • 완벽한 통합 플랫폼
    • 필요에 따라 확장 가능
    • 단순화된 클라우드 네이티브 채택

    기능

    Design modern enterprise integrations
    최신 엔터프라이즈 통합 설계
    마이크로서비스와 API 간의 엔터프라이즈 통합 패턴 및 상호 작용을 시각적으로 개발합니다.
    즉시 사용 가능한 팔레트와 엔터프라이즈 엔드포인트 및 데이터 소스에 대한 다양한 커넥터를 사용하여 애플리케이션과 데이터의 통합을 가속화합니다. 또한 Consul, Spring Cloud Config 또는 Kubernetes ConfigMaps와 같은 서비스 검색 및 구성 관리를 위해 내장된 클라우드 네이티브 도구를 사용합니다.
    Build with control and flexibility
    제어 및 유연성으로 구축
    선호하는 버전 제어 시스템에 대한 원활한 연결을 통해 변경 관리 제어를 구현합니다.
    시각적 분석기로 미사용 리소스를 식별하고 제거합니다. Docker, Cloud Foundry 및 OpenShift에 사용 가능한 스크립트를 활용하여 빌드 프로세스를 단순화하십시오. 또한 CLI 또는 Apache Maven을 사용하여 배포 아카이브 또는 Docker 이미지 빌드를 자동화합니다.
    Rigorously test integration processes
    엄격한 테스트 통합 프로세스
    통합 프로세스를 확인하는 단위 테스트를 시각적으로 설계하고, Apache Maven을 사용하여 테스트 사례 실행을 자동화하고, 테스트 실행 및 코드 적용 범위 세부 정보를 보여주는 보고서를 생성합니다.
    RESTful API에 대한 Swagger 문서를 자동으로 생성하고 Swagger 사양에 정의된 API 서비스를 테스트합니다.
    기본 제공되는 시각적 디버거를 사용하여 통합 논리에서 발견되지 않은 문제를 디버깅합니다.
    Deploy portable integrations
    휴대용 통합 배포
    하이브리드 및 다중 클라우드 환경에서 통합을 원활하게 포팅할 수 있는 유연성을 확보하십시오.
    통합을 한 번 구축하고 온프레미스, TIBCO Cloud 또는 Docker, OpenShift, Kubernetes 및 Cloud Foundry와 같은 주요 컨테이너 플랫폼이나 Amazon AWS, Microsoft Azure 및 Google Cloud Platform에서 제공하는 컨테이너 서비스에 배포하십시오.
    Manage mission critical integrations
    미션 크리티컬 통합 관리
    PaaS 및 컨테이너 플랫폼의 기본 수평 확장, 로드 밸런싱, 자가 복구 및 상태 확인 기능을 활용할 수 있는 휴대용 통합 아카이브를 배포합니다. 즉시 사용 가능한 프로세스 모니터링 및 로깅 기능 또는 개방형 API를 통해 통합할 수 있는 선호하는 도구를 활용하십시오. 또한 OpenTracing 및 Jaeger에 대한 기본 제공 지원으로 분산 추적을 구현합니다.
